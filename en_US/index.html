Description

File centipede is an internet file download/upload manager, WebDAV client, FTP client, and an SSH client. It's fast, customizable, user-friendly, multi protocols supported, and free with no ads. It also contains many useful auxiliary tools such as HTTP requester, file merge, and encoders. With the browser integration, you can download audio and videos from websites, even encrypted videos. File centipede makes everything easy and efficient, it's the best internet download manager you can find.
